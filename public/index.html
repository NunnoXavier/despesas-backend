Despesas - Documentação

Tabelas

transactions

Movimentações financeiras de débito ou crédito

Campos:

id: number;
amount: number;
description: string;
data: Date;
category: number;
account: number;

Categories

Classificação das movimentações financeiras

Campos:

id: number;
description: string;
type: string;

Accounts

Contas (bancarias ou não) onde acontecem as movimentações

Campos:

id: number;
description: string;

Métodos

  • GET/criar-tabelas

    • Método para criar as tabelas. Caso exista a tabela é ignorada

  • GET/movimentacoes

    • Retorna um objeto .json com todas as movimentações. Para selecionar um registro específico deve-se usar a query "?id=[número do id]"

  • PUT/movimentacoes

    • Insere uma movimentação. Este método recebe um objeto json no corpo da requisição no seguinte formato:
    {
    "description": "Teste 3",
    "data": "2025-02-25",
    "amount": 53.32,
    "category": 3,
    "account": 1
    }

  • PATH/movimentacoes

    • Altera um ou mais campos de uma movimentação exceto o id. Este método recebe um objeto json no corpo da requisição no seguinte formato:
    {
    "id": 1,
    "description": "Teste 3",
    "data": "2025-02-25",
    "amount": 53.32,
    "category": 3,
    "account": 1
    }

  • DELETE/movimentacoes

    • Deleta uma movimentação. Este método recebe um objeto json no corpo da requisição no seguinte formato:
    {
    "id": 1,
    }

  • GET/categorias

    • Retorna um objeto .json com todas as categorias. Para selecionar um registro específico deve-se usar a query "?id=[número do id]"

  • PUT/categorias

    • Insere uma categoria. Este método recebe um objeto json no corpo da requisição no seguinte formato:
    {
    "description": "Aluguel",
    "type": "D" ( "D" = Débito, "C" = Crédito ) }

  • PATH/categorias

    • Altera um ou mais campos de uma categoria exceto o id. Este método recebe um objeto json no corpo da requisição no seguinte formato:
    {
    "id": 1,
    "description": "Teste 3",
    "type": "C" }

  • DELETE/categorias

    • Deleta uma categoria. Este método recebe um objeto json no corpo da requisição no seguinte formato:
    {
    "id": 1,
    }

  • GET/contas

    • Retorna um objeto .json com todas as contas. Para selecionar um registro específico deve-se usar a query "?id=[número do id]"

  • PUT/contas

    • Insere uma conta. Este método recebe um objeto json no corpo da requisição no seguinte formato:
    {
    "description": "Bradesco",
    }

  • PATH/contas

    • Altera um ou mais campos de uma conta exceto o id. Este método recebe um objeto json no corpo da requisição no seguinte formato:
    {
    "id": 1,
    "description": "Itau",
    }

  • DELETE/contas

    • Deleta uma conta. Este método recebe um objeto json no corpo da requisição no seguinte formato:
    {
    "id": 1,
    }
